# All in one Converter

<b>Overview :</b> 

In this application user can do math operations addition, subtraction, multiplication , division, exponential operations, roots, trigonometric function and
 logarithmic function easily. User can easily calculate Body mass index (BMI) , convert Currency one currency to another currency & convert Temperature one scale
to another scale. This simple application also provides Multiplication Table . If you
want to use those features, you must login or need to create an account. If you forgot
your password you cam easily recover the password. User can contact with admin and
 send feedback from contact section. User can also chat with Java bot (chat bot) in this
 application. In this application to create those features I used Java Swing and AWT
 components such as menubar, JToggleButton, JFrame, JButton, JPanel, JTextField, JTextArea, JPasswordField, JComboBox, JSlider, JSpinner, JLabel and etc. Here also
used actionListener, KeyListener , MouseListener and ChangeListener.

<b>Features: </b>

i. Simple Calculator

ii. BMI Calculator

iii. Currency Converter

iv. Temperature Converter

v. Multiplication Table

vi. Chat Bot



<b>Technology:</b> Object Oriented Programming Language (Java). 

<b>Editor or IDE:</b> NetBeans IDE 8.0.2



<b>Conclusion:</b> This is a simple application that provides a Simple calculator , Body
mass index (BMI) calculator, Currency Converter, Temperature Converter and Multiplication Table.
